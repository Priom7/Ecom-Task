ECOM Task APP made with React, Redux, Laravel.

## ECOM TASK. 

git clone https://github.com/Priom7/Ecom-Task.git

cd backend 
composer install 
php artisan migrate:refresh --seed
php artisan serve

cd frontend
npm install
npm start
